**PIVOT (2026-06-15) — source = admin.cars Listings Optimizer, NOT Tableau LEI.** Some HC stores aren't in the Tableau LEI view (RLS limits it to the AE's assigned dealers). The admin.cars **Listings Optimizer → "Live Inventory" crosstab** (per dealer, no RLS) is a superset: per vehicle it carries Stock/VIN/Stock type/YMMT/**Price badge**/Price vs Market %/Days live/Photos/Price/**Reduce by**/**Good badge target**/**Great badge target**/SRPs/VDPs/Connections. Tableau custom views NO LONGER needed (task dropped). **Proven end-to-end on Honda 2026-06-15:** 98 used, 8 within $500, draft `r5315374712890474263` + sheet populated. To pull a store's LO CSV: admin.cars `/dealers/<UUID>/reports/listings_optimizer` → **Inventory** tab → **Download Crosstab** → CSV (lands `~/.playwright-mcp/Live-Inventory.csv`, rename `<key>_lo.csv`).
