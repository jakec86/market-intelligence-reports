Use correct Cars.com terminology in all dealer reports:

- **Connections** = total engagements (web transfers, email leads, phone leads, chat leads, walk-ins, chat events, other)
- **Leads** = Phone + Email + Chat only (direct contact intent subset of connections)

**Why:** Jake corrected this during the Sonic Porsche review. Kortney Boston and dealers understand "leads" as actionable contacts (phone/email/chat), not web transfers or walk-ins. Using the wrong term in a competitive review makes the data look inflated or confusing.

**How to apply:** When pulling admin.cars.com Performance Trends data, always break out both Connections (total) and Leads (P+E+C) as separate tables. Apply the same split to Cost/Connection and Cost/Lead calculations.
